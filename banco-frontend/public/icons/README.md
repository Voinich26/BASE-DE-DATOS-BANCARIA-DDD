# Iconos PWA Enterprise

Los siguientes iconos son necesarios para la PWA BancoDDD:

## Iconos PNG (requeridos para PWA)
- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

## Iconos Maskable (para Android)
- icon-maskable-192.png (192x192)
- icon-maskable-512.png (512x512)

## Iconos SVG (ya existentes)
- icon-192.svg (192x192)
- icon-512.svg (512x512)

## Generación de Iconos
Para generar los iconos PNG desde el SVG, use herramientas como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- CLI: `sharp` o `imagemagick`

## Especificaciones
- Color de fondo: #0d1b2a (slate-950)
- Color primario: #3b82f6 (blue-500)
- Estilo: Enterprise banking con shield o logo bancario
- Transparente para iconos maskable
